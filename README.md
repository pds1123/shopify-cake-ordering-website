# Shopify Cake Ordering Website

A Shopify-based cake ordering website built to explore e-commerce store setup, theme customization, and customer ordering workflows.

## Live Demo

https://vivianscakeshouse.myshopify.com/

## Project Overview

This project is a custom cake ordering website built with Shopify. The goal was to create a realistic small-business e-commerce store with custom product pages, cake ordering fields, and different product templates.

## Key Features

- Customized Shopify Dawn theme layout and storefront design
- Added custom product fields using Custom Liquid
- Created pickup date and pickup time fields for cake orders
- Added cake message and special request fields
- Created separate product templates for cakes and accessories
- Embedded Google Maps on the contact page
- Customized navigation and collection structure

## Tech Stack

- Shopify
- Dawn Theme
- Liquid
- HTML
- CSS
- JavaScript

## What I Learned

Through this project, I gained hands-on experience with Shopify theme customization, Custom Liquid, product templates, and improving customer ordering workflows for an e-commerce store.
